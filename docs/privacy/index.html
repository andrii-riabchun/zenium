Privacy Policy

Effective date: April 23, 2026

Zenium is a Chrome extension that applies visual styles to supported websites in your browser.

Information Zenium handles

Zenium processes website URLs and hostnames locally on your device to determine whether a style should be applied to the current page.

Information stored locally

Zenium stores its settings in chrome.storage.local. This includes global settings, per-website toggles, cached style definitions and mappings, the configured style repository URL, and the last successful refresh time.

Network requests

Zenium downloads style definitions from the configured repository URL. By default, this is https://sameerasw.github.io/my-internet/styles.json. If you change the repository URL in the extension settings, Zenium will send requests to the URL you provide.

Zenium does not include the pages you visit, page content, form data, or your saved settings in those requests. As with any normal web request, the host serving the style file may receive standard request metadata such as your IP address and user agent.

Data sharing

Zenium does not sell your personal data. Zenium does not transmit or share visited page URLs or hostnames, page content, or form data with the developer.

What Zenium does not collect

Zenium does not collect account information, passwords, payment information, form entries, analytics events, or advertising identifiers.
